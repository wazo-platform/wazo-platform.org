Hello Wazo Platform community!

Here is a short review of the Wazo Platform 25.12 release.

## Changelog

### Technical

- **wazo-webhookd**: improved error handling & logging for FCM push notifications
- **wazo-webhookd**: subscription lifecycle events
- **mobile/push notifications**: wazo-auth API now supports `PUT` on
  `/0.1/users/<uuid>/external/mobile` endpoint for registering mobile push tokens

For more details about the aforementioned topics, please see the
[changelog](https://wazo-dev.atlassian.net/issues/?jql=project%3DWAZO%20AND%20fixVersion%3D25.12).

See you at the next sprint review!
